# Software Developer Portfolio

## Description
A personal software developer portfolio showcasing my skills, projects, and contact details. Built using HTML, CSS, and JavaScript with a responsive design and deployed using GitHub Pages.

## Tech Stack
- HTML5
- CSS3
- JavaScript

## Features
- Responsive design (mobile, tablet, desktop)
- Clean and simple UI
- Projects with descriptions
- Contact information

## How to Run Locally
1. Download or clone the repository
2. Open `index.html` in any web browser

## Live Demo
This portfolio is deployed using GitHub Pages.
